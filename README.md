# Team Terka Client (Azure Training Project)

### NextJS Client App

### Requirements 

* min node version 18.17 - ```nvm use 18.17```

### Clone repository

```git clone git@github.com:EsetHQ/azure-training-team-terka-client.git```

### Project installation

```npm install```

### Start development server

```npm run dev```
